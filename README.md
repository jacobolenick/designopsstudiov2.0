# DesignOps

Product design and design systems studio — landing page for rebranded business.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the site.

## Build

```bash
npm run build
npm run preview
```

## Customize

- Update contact email in `src/components/Footer.tsx`
- Adjust copy in each section component under `src/components/`
- Colors and fonts are defined in `tailwind.config.js`
